XXE Injection (XML External Entity)

Vysvetlenie

XXE Injection zneužíva nebezpečne nastavený XML parser, ktorý umožní načítať externé entity (súbory alebo URL). Takto môže útočník čítať súbory zo servera alebo spôsobiť DoS útok.

Skús si príklad

Vlož sem jednoduchý XML dokument a klikni „Spracovať XML“. Potom skús vložiť príklad s entitou ako: 

<!DOCTYPE data [
  <!ENTITY xxe SYSTEM "file:///etc/passwd">
]>
<root>&xxe;</root>
