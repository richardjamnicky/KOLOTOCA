XXE Injection (XML External Entity)

XXE útok zneužíva parsovanie XML dát, kde je povolené načítanie externých entít. Útočník tým môže čítať citlivé súbory zo servera alebo spôsobiť DoS útok.

Príklad

<!DOCTYPE foo [ <!ENTITY xxe SYSTEM "file:///etc/passwd"> ]>
<data>&xxe;</data>

Ochrana

  • Zakáž parsovanie externých entít v XML parseri.
  • Validuj všetky vstupné XML dáta.
  • Používaj bezpečné knižnice.
