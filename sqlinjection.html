SQL Injection

SQL Injection nastáva, keď aplikácia vkladá používateľský vstup priamo do SQL príkazu bez správnej validácie. Útočník tak môže manipulovať dotaz a získať prístup k databáze.

Príklad

SELECT * FROM users WHERE username = 'admin' AND password = '';
  OR '1'='1';

Ak aplikácia tento vstup nesprávne spracuje, môže útočník získať zoznam všetkých používateľov.

Ochrana

  • Používaj parameterizované dotazy alebo ORM.
  • Vždy validuj a escapuj vstupy.
  • Nepoužívaj string concatenation v SQL dotazoch.
